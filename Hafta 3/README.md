# Web Programlama Hafta 3

## Amaç

Bu klasör, temel HTML yapısını ve sayfa elemanlarının kullanımını örnekler üzerinden göstermek amacıyla hazırlanmıştır. Çalışmalar, web sayfası oluşturmanın başlangıç düzeyindeki yapı taşlarını göstermektedir.

## Çalışma Biçimi

Her HTML dosyası bağımsız bir örnek olarak düzenlenmiştir. Dosyalar tarayıcıda açıldığında ilgili sayfa yapısı doğrudan görüntülenir.

## Kullanım

İncelenmek istenen HTML dosyası bir web tarayıcısı ile açılarak çalıştırılır.
