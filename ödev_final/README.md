# Yapılacaklar Listesi Uygulaması

## Amaç

Bu proje, kullanıcının yapılacak işlerini liste halinde ekleyip takip edebilmesini sağlayan basit bir web uygulamasıdır. Çalışma, web arayüzü üzerinde görev ekleme, filtreleme ve temizleme işlemlerini göstermektedir.

## Çalışma Biçimi

Uygulama HTML, CSS, Bootstrap ve JavaScript ile hazırlanmıştır. Kullanıcı metin alanına görev girer, ekleme düğmesiyle listeye aktarır ve görevleri durumlarına göre filtreleyebilir.

## Kullanım

`index.html` dosyası tarayıcıda açılarak uygulama çalıştırılır. Görevler arayüz üzerinden eklenir, filtrelenir veya temizlenir.
