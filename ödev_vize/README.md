# Kişisel Haber Web Sitesi

## Amaç

Bu proje, haber içeriklerinin kategori temelli bir web sitesi yapısında sunulmasını amaçlamaktadır. Ekonomi, sağlık, spor ve teknoloji gibi başlıklar altında farklı sayfalar oluşturulmuştur.

## Çalışma Biçimi

Ana sayfa ve kategori sayfaları HTML ile hazırlanmıştır. Sayfa görünümü CSS dosyası ile düzenlenmiş, görsel içerikler `img` klasörü üzerinden kullanılmıştır.

## Kullanım

`index.html` dosyası tarayıcıda açılarak ana sayfa görüntülenir. Diğer kategori sayfalarına bağlantılar üzerinden geçiş yapılabilir.
