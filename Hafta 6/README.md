# Web Programlama Hafta 6

## Amaç

Bu klasör, JavaScript ile web sayfalarına temel etkileşim kazandırma çalışmalarını içerir. Örnekler, istemci tarafında çalışan komutların sayfa davranışını nasıl etkilediğini göstermektedir.

## Çalışma Biçimi

HTML dosyaları sayfa yapısını, JavaScript dosyası ise sayfa üzerinde çalışan komutları içerir. Örnekler tarayıcı üzerinde çalıştırıldığında JavaScript işlemleri gözlemlenebilir.

## Kullanım

HTML dosyaları tarayıcıda açılır. JavaScript kodları `scriptKomutları.js` dosyası ve ilgili HTML bağlantıları üzerinden incelenebilir.
