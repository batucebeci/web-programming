# Web Programlama Hafta 10

## Amaç

Bu klasör, Bootstrap kullanarak duyarlı web arayüzleri oluşturma çalışmalarını içerir. Amaç, hazır arayüz bileşenleriyle düzenli ve uyumlu sayfa yapıları kurmayı göstermektir.

## Çalışma Biçimi

HTML dosyaları, Bootstrap dosyaları ve jQuery ile birlikte çalışır. Örnekler tarayıcıda açıldığında Bootstrap bileşenlerinin sayfa düzenine etkisi incelenebilir.

## Kullanım

İlgili HTML dosyası tarayıcıda açılarak çalıştırılır. Bootstrap klasörünün ve diğer yardımcı dosyaların aynı dizin yapısında kalması gerekir.
