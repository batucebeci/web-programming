# Web Programlama Hafta 7

## Amaç

Bu klasör, web programlama dersine ait etkileşimli sayfa örneğini içermektedir. Çalışma, HTML ve JavaScript yapılarının birlikte kullanımını göstermeyi amaçlar.

## Çalışma Biçimi

Örnek dosya tarayıcıda çalıştırıldığında sayfa üzerinde tanımlanan davranışlar doğrudan gözlemlenir. Kod yapısı, sayfa içeriği ile işlem mantığının birlikte nasıl kullanılabileceğini gösterir.

## Kullanım

`ornek1.html` dosyası tarayıcıda açılarak uygulama çalıştırılır.
