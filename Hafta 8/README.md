# Web Programlama Hafta 8

## Amaç

Bu klasör, web sayfalarında görsel içerik kullanımı ve etkileşimli örnekler üzerine hazırlanmış çalışmaları içerir. Amaç, sayfa içeriğinin görseller ve temel betik yapılarıyla desteklenmesini göstermektir.

## Çalışma Biçimi

HTML dosyaları, ilgili görsel dosyalarla birlikte çalışır. Örnekler tarayıcıda açıldığında sayfa düzeni ve görsel kullanım biçimi incelenebilir.

## Kullanım

HTML dosyaları tarayıcıda açılarak çalıştırılır. Görsel dosyaların HTML dosyalarıyla aynı klasör yapısı içinde kalması gerekir.
