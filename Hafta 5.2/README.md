# Web Programlama Hafta 5.2

## Amaç

Bu klasör, HTML ve CSS ile temel sayfa tasarımı oluşturma çalışmalarını içerir. Amaç, web arayüzlerinde içerik yerleşimi ve biçimlendirme yaklaşımını uygulamalı olarak göstermektir.

## Çalışma Biçimi

Her örnek dosya, belirli bir sayfa düzeni veya biçimlendirme yapısını göstermektedir. Dosyalar tarayıcıda doğrudan çalıştırılabilir.

## Kullanım

İlgili HTML dosyası web tarayıcısı ile açılarak çalışma sonucu görüntülenir.
