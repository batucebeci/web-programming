# Web Programlama Hafta 9

## Amaç

Bu klasör, jQuery kullanımı ile web sayfalarında etkileşim oluşturma çalışmalarını içerir. Örnekler, hazır JavaScript kütüphaneleriyle sayfa elemanları üzerinde işlem yapmayı göstermektedir.

## Çalışma Biçimi

HTML dosyaları, klasörde yer alan jQuery dosyası ile birlikte çalışır. Her örnek, farklı bir etkileşim veya sayfa davranışını gösterecek şekilde hazırlanmıştır.

## Kullanım

İlgili HTML dosyası tarayıcıda açılarak çalıştırılır. jQuery bağlantısının doğru çalışması için klasör yapısı korunmalıdır.
