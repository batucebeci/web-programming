# Web Programlama Hafta 4

## Amaç

Bu klasör, HTML ile sayfa içeriği oluşturma ve farklı sayfa bölümlerini düzenleme konularına ait örnekleri içerir. Çalışmalar, web sayfası yapısının daha düzenli biçimde kurulmasını amaçlamaktadır.

## Çalışma Biçimi

Her örnek dosya, belirli bir HTML kullanımını gösterecek şekilde hazırlanmıştır. Dosyalar birbirinden bağımsız olarak açılıp incelenebilir.

## Kullanım

İlgili HTML dosyası tarayıcıda açılarak sayfa çıktısı görüntülenir.
