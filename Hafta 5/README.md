# Web Programlama Hafta 5

## Amaç

Bu klasör, web sayfalarında biçimlendirme ve görsel düzen oluşturma konularına ait örnekleri içerir. Çalışmalar, HTML yapısının stil kuralları ile nasıl desteklenebileceğini göstermektedir.

## Çalışma Biçimi

Örnek dosyalar, farklı sayfa düzeni ve biçimlendirme uygulamalarını ayrı ayrı göstermektedir. Her dosya tarayıcı üzerinde çalışacak şekilde hazırlanmıştır.

## Kullanım

İncelenmek istenen HTML dosyası tarayıcıda açılır. Sayfa düzeni ve kullanılan stil yapıları dosya içeriği üzerinden incelenebilir.
