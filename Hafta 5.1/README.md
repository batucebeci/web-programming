# Web Programlama Hafta 5.1

## Amaç

Bu klasör, CSS kullanımı ile web sayfalarında görsel düzen oluşturmayı göstermektedir. Çalışma, HTML içeriğinin ayrı bir stil dosyası ile nasıl biçimlendirilebileceğini örnekler.

## Çalışma Biçimi

HTML dosyaları sayfa içeriğini, CSS dosyası ise sayfanın görünüm kurallarını içerir. Dosyalar birlikte çalışarak tarayıcıda biçimlendirilmiş bir sayfa çıktısı üretir.

## Kullanım

HTML dosyaları tarayıcıda açılarak çalıştırılır. Stil kuralları `still.css` dosyası üzerinden incelenebilir.
