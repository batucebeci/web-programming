# Web Programlama Uygulamaları

## Amaç

Bu çalışma, web programlama dersi kapsamında hazırlanan temel uygulama örneklerini içermektedir. HTML, CSS, Bootstrap ve temel web sayfası yapıları üzerinde yapılan alıştırmalar bir araya getirilmiştir.

## Çalışma Biçimi

Klasörlerde haftalara göre ayrılmış örnekler bulunmaktadır. Her örnek, ilgili konunun sayfa yapısı, biçimlendirme veya temel arayüz mantığını göstermek amacıyla hazırlanmıştır.

## Kullanım

İncelenmek istenen klasördeki HTML dosyası bir web tarayıcısı ile açılarak çalıştırılabilir. Bootstrap kullanılan örneklerde ilgili CSS ve JavaScript dosyalarının aynı klasör yapısı içinde bulunması gerekir.
